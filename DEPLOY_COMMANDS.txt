### DEPLOY (Cloudflare Pages über GitHub Actions) ###

Stand: 25.02.2026

1) Änderungen vormerken:
git add .

2) Änderungen committen:
git commit -m "Kurz beschreiben, was geändert wurde"

3) Nach GitHub pushen:
git push origin main

Danach startet automatisch der Workflow
`.github/workflows/deploy.yml`.

Ziel ist Cloudflare Pages (Projekt: `mindfulpractice-app`).
