Hier sind die universellen Befehle für dein Terminal, um DEINE Änderungen auf GitHub hochzuladen:

Option 1: Alles in einer Zeile (Schnellste Methode)
git add . && git commit -m "Hier deine Beschreibung eintragen" && git push origin main

Option 2: Die Befehle einzeln (Schritt für Schritt)
1. Änderungen sammeln: 
   git add .

2. Änderungen bestätigen: 
   git commit -m "Hier beschreiben, was du geändert hast"

3. Zu GitHub hochladen: 
   git push origin main

WICHTIG: Ersetze den Text in den Anführungszeichen immer durch eine kurze Info, was du getan hast (z.B. "Texte korrigiert" oder "Farben angepasst").
